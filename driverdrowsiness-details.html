Driver Safety Assistant

AI-Powered Road Safety

Company: Independent Project Role: Developer
The Anti-Drowsiness Driver Assistant is an AI-powered system designed to enhance road safety by detecting driver drowsiness in real-time and providing alerts. It integrates intelligent suggestions for nearby amenities and comprehensive data collection to reduce drowsiness-related accidents by 25%.

Key Contributions

  • Real-time drowsiness detection using OpenCV and AI models.
  • 25% reduction in drowsiness-related risks.
  • Intelligent suggestions for nearby amenities.
  • Comprehensive data collection for improved assistance.
  • Cross-platform interface with Flutter.

Technologies Used

Python OpenCV Flutter AI Model Training Data Analysis
