*SETUP*
-------
To run this portfolio locally:

*Prerequisites:*
- Modern web browser (e.g., Chrome, Firefox)
- Optional: Node.js for a local server (e.g., live-server)

*Installation:*
1. Clone the repository:
   git clone https://github.com/JaskaranChana/[your-repo-name].git
   cd [your-repo-name]
2. Open the site:
   - Directly in a browser: open index.html
   - Or via a local server:
     npm install -g live-server
     live-server

*Dependencies:*
- Bootstrap 5
- AOS (Animate on Scroll)
- Typed.js
- GLightbox
- Swiper.js
(All vendor files are in assets/vendor/)
